# Valheim Climbing Mod

This mod adds climbing mechanics like in BoTW to Valheim !

## Building

1. Open the solution in Visual Studio or JetBrains Rider.
2. Ensure you have the necessary references (Valheim assemblies, BepInEx, Harmony).
3. Create your own Environment.props from the template.
3. Build the project.

## Usage

Go to https://thunderstore.io/c/valheim/p/Rotceh999/ClimbingMod/ to learn more about my mod, and install it if you want.
If you want to install it manually :
1. You will need to install BepInEx
2. download the "ClimbingMod" zip folder and extract it in your plugins folder (from Steam folder : Steam\ steamapps\ common\ Valheim\ BePinex\ plugins)
3. Play and tweak the values in the config file.
